# SnipSpark
# CSS Snippet Viewer (Flask + OOP)

Simple student project: a web app to view, copy and preview CSS snippets.

## Features
- Add CSS snippets (title, description, css code)
- Preview snippet applied to a sample element (live preview)
- Copy snippet code to clipboard
- Snippets stored in `snippets.json` so no DB needed
- Uses Type-safe, textbook-style OOP using Python classes (Snippet + SnippetManager)
